# 家庭时光局品牌调整规格

## 目标

将长期使用的提醒工具从“老爸时光局”调整为父母都适用的“家庭时光局”，同时保留本次为爸爸生日准备的旅行惊喜。

## 文案边界

- 通用品牌名统一为“家庭时光局”：网页标题、提醒页眉、设置页、备份文件、README 与仓库说明。
- 生日流程继续以爸爸为祝福对象，保留“爸，生日快乐”等只属于本次生日惊喜的文字。
- 旅行册工具栏改为“家庭旅行册”，礼物卡品牌改为“家庭时光局”。
- 底层 LocalStorage 旧键继续兼容，避免现有提醒和已播放状态丢失；新增数据仍沿用兼容键。
- ICS UID 域改为 `family-time-station`，不影响提醒内容。

## 部署

- 源代码发布到公开仓库 `a235927015/family-time-station`。
- 使用 GitHub Actions 构建 Vite 静态站并部署 GitHub Pages。
- 预期公开地址为 `https://a235927015.github.io/family-time-station/`。

## 验收

- 通用界面不再出现“老爸时光局”或“老爸旅行册”。
- 爸爸生日祝福仍可见。
- 旧 LocalStorage 数据可继续读取。
- 全部自动化测试与生产构建通过。
- GitHub Pages 公开地址返回成功页面。
