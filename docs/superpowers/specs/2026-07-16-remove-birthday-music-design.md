# 生日旅行册取消音乐设计

## 决策

有声自动播放在移动浏览器中通常需要用户先点击页面，无法保证微信和 iPhone 打开即播放。因此生日旅行册完全取消音乐，不提供自动播放或手动播放。

## 范围

- 不渲染 `<audio>` 元素，也不请求任何音频资源。
- 移除“播放音乐 / 关闭音乐”按钮及相关状态和逻辑。
- 旅行册翻页、跳过、生日祝福、首次自动展示和设置页重复观看均保持不变。
- 删除音频占位说明，避免后续误以为仍需上传音乐。

## 验收

- 生日旅行册页面不存在音频元素和音乐按钮。
- 首次打开仍显示生日旅行册。
- 看完后再次打开进入提醒页。
- 设置页“重看生日旅行册”仍可重复观看。
- 自动化测试、生产构建和 GitHub Pages 部署通过。
