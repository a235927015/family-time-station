# 家庭时光局

一个为爸爸生日准备、也适合爸妈长期使用的移动网页。第一次打开会播放“旅行胶片册”，随后进入家庭周期提醒工具。

公开访问地址：<https://a235927015.github.io/family-time-station/>

## 已实现

- 10 站旅行照片生日册，支持滑动、按钮翻页、暂停、静音和跳过。
- 一次性、每周、每月和每年提醒。
- 本周事项、逾期事项、完成当前一期、修改和删除。
- 每月 29、30、31 日遇到短月份时按当月最后一天处理。
- 提醒导出为 ICS 文件，可加入手机系统日历。
- 本地 JSON 备份和恢复。
- 数据只保存在当前手机，不需要登录或服务器。

## 本地运行

```powershell
npm.cmd install
npm.cmd run dev -- --host 0.0.0.0
```

终端会显示电脑局域网地址。同一 Wi-Fi 下的手机可访问该地址。正式送出时应部署到 HTTPS 静态网站。

## 测试与构建

```powershell
npm.cmd test
npm.cmd run build
```

构建结果在 `dist/`，可部署到 Vercel、Netlify、Cloudflare Pages 或 GitHub Pages。项目使用相对资源路径，可部署在网站根目录或仓库子目录。

## 替换旅行内容

1. 从照片中精选 10 张，压缩到每张约 500KB 内。
2. 依次命名为 `01.jpg` 至 `10.jpg`，放入 `public/trips/`。
3. 在 `src/features/birthday/trips.ts` 中修改每张照片的地点和一句回忆。
4. 如果人物在手机竖屏中被裁切，为对应站点增加 `objectPosition`，例如 `objectPosition: '50% 30%'`。
5. 把合法可用的音乐命名为 `birthday.mp3`，放入 `public/audio/`。
6. 再次运行测试与构建。

照片或音乐缺失时，程序会显示清晰的占位画面，不会出现破损图片图标或阻断后续功能。

## 手机日历说明

“加入手机日历”会下载标准 `.ics` 文件。普通手机浏览器通常可直接打开；微信内置浏览器可能阻止下载，此时点右上角菜单，选择“在浏览器打开”后再加入。

## 数据与换手机

提醒保存在浏览器 LocalStorage 中。爸爸和妈妈各自手机的数据互不自动同步。换手机或清理微信缓存前，应在“设置”中导出 JSON 备份，再在新手机导入。

## GitHub 仓库与部署

源代码保存在公开仓库 `a235927015/family-time-station`。推送到 `main` 后，GitHub Actions 会自动运行测试、构建并部署 GitHub Pages。

## 交付前实机检查

- Android 微信打开、翻完生日册并新增提醒。
- iPhone 微信打开、翻完生日册并新增提醒。
- 两台手机分别测试 ICS 下载；若微信阻止下载，确认提示流程可理解。
- 检查所有真实照片的人脸裁切和音乐音量。
